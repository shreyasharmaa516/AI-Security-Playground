# Sentra

> AI Security Evaluation Platform

## Overview

Sentra is an open-source platform for evaluating the security of AI-powered applications through safe, defensive testing.

The goal is to help developers and security enthusiasts understand common AI security risks through controlled simulations and clear reporting.

## Planned Features

- Prompt Injection Testing
- Jailbreak Detection
- AI Response Evaluation
- Security Reports
- Interactive Dashboard
- Extensible Plugin Architecture

## Tech Stack

- Python
- FastAPI
- SQLite
- Docker
- Ollama (Local LLM)
- HTML/CSS/JavaScript (Version 1)
